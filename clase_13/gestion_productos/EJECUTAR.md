# 🚀 Instrucciones Rápidas para Ejecutar

## 1. Instalar dependencias

```bash
pip install -r requirements.txt
```

## 2. Ejecutar la aplicación

```bash
python app_productos.py
```

## 3. Abrir en el navegador

```
http://localhost:5000
```
